01 capabilities

GitHub & URLs

Install from GitHub release assets or any direct download URL, verified with SHA-256 checksums.

TOML manifest

Declare the tools you want in a plain config file. One command reproduces your whole toolbox.

Embedded registry

Curated recipes are compiled into the binary, so paq works fully offline — with optional live updates.

Latest & upgrade

Pin exact versions or track latest; paq upgrade rolls everything forward cleanly.

Six targets

Pre-built binaries for linux, macOS and windows on both amd64 and arm64.

Self-updating

paq self-update keeps the tool current; paq doctor checks your PATH and setup.
