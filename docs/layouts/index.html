01 what paq is

Fast to install

One command drops a tool or application straight onto your machine — no build step, no package database to prime, no daemon to wait on.

Never invasive

paq installs into plain directories you own and keeps its state to itself. No runtime, no shell hijacking — it never takes over your environment.

Versions, your call

paq won't switch active versions behind your back. It doesn't manage them for you — but running several side by side is trivial, each in its own place.

02 capabilities

GitHub & URLs

Install from GitHub release assets or any direct download URL, verified with SHA-256 checksums.

TOML manifest

Declare the tools you want in a plain config file. One command reproduces your whole toolbox.

Embedded registry

Curated recipes are compiled into the binary, so paq works fully offline — with optional live updates.

Latest & upgrade

Pin exact versions or track latest; paq upgrade rolls everything forward cleanly.

Side-by-side versions

Keep several versions of the same tool at once — each install goes to its own dest, so nothing collides and nothing else is touched.

Self-updating

paq self-update keeps the tool current; paq doctor checks your PATH and setup.
